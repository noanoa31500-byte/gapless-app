# ビッグデータ可視化システム 技術解説

## 🎯 なぜこのシステムが命を守るのか

### 従来のハザードマップの課題

**静的な情報**:
- 「ここが浸水する可能性があります」という範囲のみ
- 水深や流速の情報が欠如
- どこを通れば安全かが分からない

**避難者の混乱**:
- 地図を読むのに時間がかかる
- パニック状態で正確な判断ができない
- 結果として危険なルートを選んでしまう

---

### 本システムの革新性

## 1. ビッグデータ連動（2000地点の予測値）

**リアルタイム可視化**:
```
satun_flood_prediction.json
├─ 2000地点の予測データ
├─ 各地点ごとに
│  ├─ 水深（pred_depth）
│  ├─ 流速（pred_speed）
│  └─ リスクスコア（risk_score）
```

**視覚的表現**:
- **濃い青（risk 100 / 水深>2m）**: 絶対に避けるべき
- **水色（risk 20-80 / 水深0.3-1.8m）**: 通行困難だが避難可能
- **半径50m/40m**: データ点を繋ぎ、面として認識可能

---

## 2. 感電リスクの可視化（黄色い警告）

**見えない死への対策**:
洪水時の感電死は、濁った水の中で電線が見えないために発生します。

**視覚化の効果**:
```
黄色い円（半径20m）= 電力設備からの危険範囲
↓
青い浸水域と重なる = 最も危険なゾーン
↓
一目で「近づいてはいけない」と理解できる
```

---

## 3. 直感的な色設計

### 色の心理学を活用

**青色（水）**:
- 人間は本能的に青色を水と認識
- 濃淡で危険度を直感的に理解

**黄色（警告）**:
- 世界共通の警告色
- 黒と黄色の組み合わせは最も視認性が高い
- オレンジの縁取りで緊急性を強調

**重なり部分（青+黄）**:
- 視覚的に「最も危険」と認識される
- 言葉がなくても理解できる

---

## 4. 技術的優位性

### パフォーマンス最適化

**Isolate処理（compute使用）**:
```dart
// 2000件のデータをバックグラウンドで処理
final floodData = await compute(_parseFloodData, jsonString);
```
- UIスレッドをブロックしない
- スムーズな地図操作を維持

**メモリ効率**:
- 不要なデータ（risk_score 0）は描画しない
- 半径を最適化してオーバーラップを最小化

---

## 5. 実際の避難シーンでの効果

### パニック状態でも使える

**従来**:
1. ハザードマップを開く
2. 現在地を確認
3. 浸水範囲を読み取る
4. 安全なルートを考える
5. → **時間がかかる、迷う**

**本システム**:
1. 地図を開く
2. → **青い場所は避ける**
3. → **黄色い場所は絶対避ける**
4. → **白いエリアを通る**
5. → **直感的、即座に判断**

---

## 6. データ駆動型の避難支援

### 予測データの活用

**水深による分類**:
- `> 2.0m`: 流される危険、車両水没
- `0.3m - 1.8m`: 歩行困難、転倒リスク
- `< 0.3m`: 通行可能だが注意が必要

**流速による分類**:
- 激流エリアは濃い青で強調
- 避難者が「ここは危険」と判断できる

---

## 7. レイヤーの重ね順戦略

### 視覚的優先順位

```
上層（最も目立つ）
│
├─ 🟡 電力設備リスク（黄色）
│   └─ 最優先で避けるべき
│
├─ 🔵 浸水リスク（青色濃淡）
│   └─ 次に避けるべき
│
└─ その他の情報
```

**重なり効果**:
- 青い海の中に黄色い警告が浮かぶ
- 最も危険な場所が一目で分かる

---

## 結論: このシステムが命を守る理由

### 3つの革新

1. **ビッグデータ + AI予測**
   - 2000地点の詳細データ
   - 水深・流速を考慮

2. **直感的な視覚化**
   - 言葉不要
   - パニック状態でも理解可能

3. **パフォーマンス最適化**
   - リアルタイム表示
   - スムーズな操作

### 防災エンジニアとしての信念

> 「最も優れた防災システムは、
> 使い方を説明する必要がないシステムである」

地図を開いた瞬間に、
**青い海**を避け、
**黄色い警告**を避け、
**白い安全地帯**へ向かう。

それだけで命が救われる。
それがこのシステムの目指す世界です。
